# ProTrader Terminal

ProTrader Terminal is a premium, high-frequency FinTech trading dashboard engineered for precision, speed, and clarity. It provides traders with a sophisticated suite of tools for position sizing, comprehensive trade journaling, market analysis, and account growth projection.

## Core Features

- **Dashboard**: A comprehensive overview of trading performance, equity curves, and recent activities.
- **Position Sizing Calculator**: Dynamically calculates lot sizes based on account balance, risk percentage, stop-loss points, and risk-to-reward ratio for specific instruments (e.g., XAUUSD).
- **Trade Journaling System**: 
  - **Trade Log**: Record individual trades including result, profit/loss, setup pattern, session, and asset. Built-in daily loss limits logic.
  - **Playbook**: Define and review trading strategies.
  - **Tags & Notes**: Categorize trades and maintain detailed session notes.
  - **Reports & Stats**: View analytical breakdowns of trading performance (win rate, total PnL, pattern success) generated by a backend NumPy framework.
- **Charts**: View and analyze market data.
- **Economic Calendar**: Stay updated on fundamental events.
- **Roadmap & Projections**: Forecast account growth over time and view income projections across various funded account tiers.

## Architecture

- **Frontend**: Built with HTML, vanilla JavaScript, and CSS. The UI adheres to a "Deep Dark" design system leveraging Glassmorphism and minimal high-contrast accents for an institutional feel.
- **Backend**: A lightweight Python HTTP server (`server.py`) handling API requests, file serving, and data processing.
- **Data Processing**: Heavy lifting and statistical calculations are performed via `trading_dashboard.py`, which utilizes NumPy for quantitative analysis.

## Getting Started

### Prerequisites

- Python 3.x installed on your system.
- Standard Python libraries (`http.server`, `socketserver`, `json`, `urllib.parse`, `os`, `sys`).
- NumPy (required for statistical calculations and projections in `trading_dashboard.py`).

### Running the Application

1. Clone or navigate to the project directory.
2. Ensure you have the necessary dependencies installed (`pip install numpy`).
3. Start the backend server by running:
   ```bash
   python server.py
   ```
4. The server will start on port `9090` by default. Open your web browser and navigate to:
   ```
   http://localhost:9090
   ```

## API Endpoints

The Python server provides the following core API routes to support the frontend:

- `GET /api/stats`: Retrieves overall and daily trading statistics.
- `GET /api/projections`: Returns funded income projections for various account sizes.
- `POST /api/trades/log`: Logs a new trade and validates daily stop limits.
- `POST /api/calculate`: Returns calculated position sizing metrics.
- `POST /api/project`: Calculates long-term compound growth projections based on initial balance and targeted monthly return.

## Design Philosophy

The application follows the "ProTrader Terminal" design specification, prioritizing data density and a "Controlled Power" emotional response. It features a dark slate canvas (`#09090b`), translucent charcoal surface panels, and strategic use of color (Electric Blue for actions, Emerald for gains, Crimson for losses) to maintain focus during extended trading sessions. Inter is utilized as the primary typeface for its clinical precision and monospaced tabular numerals.
